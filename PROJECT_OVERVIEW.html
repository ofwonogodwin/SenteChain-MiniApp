üöÄ Quick Start

# Clone the repository
git clone <repo-url>
cd sentechain-miniapp

# Run setup script
./setup.sh

# Compile contracts
npm run compile

# Deploy to Base Sepolia
npm run deploy

# Start development
npm run dev

Then open http://localhost:3000 in your browser!
