💻 TERMINAL COMMANDS

DEVELOPMENT:
  npm run compile          → Compile smart contracts
  npm run deploy           → Deploy to local network
  npm run deploy:base      → Deploy to Base Sepolia
  npm run dev              → Start all services
  npm test                 → Run tests

INDIVIDUAL SERVICES:
  cd backend && npm start  → Start backend only
  cd frontend && npm run dev → Start frontend only

HARDHAT:
  npx hardhat node         → Start local blockchain
  npx hardhat console      → Open Hardhat console
  npx hardhat test         → Run contract tests

  Frontend not loading?

  Backend connection error?

  Contracts not found?
    → npm run compile && npm run deploy

  MetaMask issues?
    → Check network (Hardhat Local or Base Sepolia)
    → Clear MetaMask cache in Settings > Advanced
